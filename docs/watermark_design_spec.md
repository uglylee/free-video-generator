# 水印设计规格 — 语言感知版

> **日期**：2026-06-30
> **目标**：根据用户 prompt 语言自动选择中/英文水印，字号自适应视频分辨率，统一视觉风格

---

## 一、多语言水印内容

| 语言 | 触发条件 | 第一行（归属行） | 第二行（地址行） |
|------|---------|----------------|----------------|
| 🇨🇳 中文 | prompt 含 CJK 字符 | `由 Agnes Video Generator 生成` | `video.lichuanyang.top` |
| 🇬🇧 英文 | prompt 纯拉丁字符 | `Generated by Agnes Video Generator` | `video.lichuanyang.top` |
| 🔀 双行（可选） | `watermark_language = "bilingual"` | `由 Agnes Video Generator 生成` | `Generated by Agnes Video Generator` |

> **设计思路**：归属行用用户语言建立信任感，地址行保持 URL 不变。

## 二、语言检测算法

```python
import re

def detect_watermark_language(text: str) -> str:
    """根据用户输入文本检测水印语言。
    
    Returns: "zh" | "en"
    """
    if not text:
        return "en"  # 默认英文
    
    # 检测 CJK 字符（中文/日文/韩文）
    cjk_pattern = re.compile(r'[\u4e00-\u9fff\u3400-\u4dbf\uf900-\ufaff]')
    if cjk_pattern.search(text):
        return "zh"
    return "en"
```

### 各管线检测文本来源

| 管线 | 检测来源字段 | 示例 |
|------|------------|------|
| Simple | `state.prompt` | `"一只猫在花园里追蝴蝶"` → zh |
| Creative | `state.idea` | `"A space adventure story"` → en |
| Manuscript | `state.manuscript_text` | `"这是第一段测试文本..."` → zh |
| Anchor | `state.script_text` | `"Welcome to today's news..."` → en |

### 检测时机

在水印后处理步骤中，读取任务状态中的对应字段进行检测，实时决定水印语言。这样修改 prompt 重新生成时语言也会随之更新。

## 三、字体选择

| 用途 | 字体文件 | 说明 |
|------|---------|------|
| CJK + 拉丁混排 | `resource/fonts/STHeitiMedium.ttc` | 项目已有，中英文渲染质量均高 |
| 纯英文环境（备选） | 系统 Helvetica / Arial | 仅在 STHeitiMedium 不可用时回退 |

> **决定**：统一使用 `STHeitiMedium.ttc`。该字体对中英文均有良好支持，避免字体不一致导致的水印跳跃感。

## 四、字号自适应计算

### 公式

```
font_size = max(12, round(video_height × 0.022))

             最小保护            画面高度比例
```

## 五、位置与间距

### 定位策略

水印固定在**右下角**，使用 ffmpeg drawtext 的相对坐标：

```
x = w - tw - right_margin
y = h - th - bottom_margin
```

### 边距公式

```
right_margin  = max(16, round(video_width  × 0.02))
bottom_margin = max(12, round(video_height × 0.015))
```

### 各分辨率边距对照

| 预设 | 分辨率 | right_margin | bottom_margin |
|------|--------|:---:|:---:|
| Portrait | 768 × 1152 | 16px | 17px |
| Landscape | 1152 × 768 | 23px | 12px |
| Square | 1024 × 1024 | 20px | 15px |

### 布局示意图

```
┌──────────────────────────────────────────────────┐
│                                                  │
│                                                  │
│                                      ┌──────────┐│
│                                      │ 归属行    ││  ← font_size
│                                      │ 地址行    ││  ← font_size
│                                      └──────────┘│
│                                      ← margin → ↑
└──────────────────────────────────────────────────┘
                                     ← margin →
```

> **为什么右下角**：视频内容的视觉重心通常在中间或左上（文字/人物），右下角是干扰最小的位置，也是行业惯例（抖音水印、YouTube 自动字幕均在此位置）。

## 六、视觉样式规范

### 配色

| 元素 | 颜色 | 不透明度 | ffmpeg 参数 |
|------|------|:---:|------|
| 文字 | 白色 `#FFFFFF` | 85% | `fontcolor=white@0.85` |
| 背景框 | 黑色 `#000000` | 35% | `box=1:boxcolor=black@0.35` |
| 文字阴影 | 黑色 `#000000` | 40% | `shadowx=1:shadowy=1:shadowcolor=black@0.4` |

### 背景框设计

半透明黑底圆角矩形，包裹两行文字：

```
┌──────────────────────────────────┐
│  boxborderw = font_size × 0.4    │  ← 内边距
│  ┌────────────────────────────┐  │
│  │ 由 Agnes Video Generator 生成│  │  ← 归属行
│  │ video.lichuanyang.top       │  │  ← 地址行 (地址行字号 = font_size × 0.85)
│  └────────────────────────────┘  │
│                                  │
└──────────────────────────────────┘
```

> **注**：地址行字号缩小 15%，形成主次层次感，避免与归属行视觉竞争。

### 地址行字号

```
url_font_size = max(10, round(font_size × 0.85))
```

| 预设 | font_size | url_font_size |
|------|:---:|:---:|
| Portrait | 25px | **21px** |
| Landscape | 17px | **14px** |
| Square | 22px | **19px** |

## 七、ffmpeg 命令模板

### 中文水印（Portrait 768×1152 为例）

```bash
ffmpeg -i input.mp4 \
  -vf "drawtext=fontfile=resource/fonts/STHeitiMedium.ttc: \
       text='由 Agnes Video Generator 生成\nvideo.lichuanyang.top': \
       fontsize=25:fontcolor=white@0.85: \
       box=1:boxcolor=black@0.35:boxborderw=10: \
       x=w-tw-16:y=h-th-17: \
       shadowx=1:shadowy=1:shadowcolor=black@0.4: \
       line_spacing=4" \
  -c:v libx264 -preset fast -crf 18 \
  -c:a copy \
  output.mp4
```

### 英文水印（Landscape 1152×768 为例）

```bash
ffmpeg -i input.mp4 \
  -vf "drawtext=fontfile=resource/fonts/STHeitiMedium.ttc: \
       text='Generated by Agnes Video Generator\nvideo.lichuanyang.top': \
       fontsize=17:fontcolor=white@0.85: \
       box=1:boxcolor=black@0.35:boxborderw=7: \
       x=w-tw-23:y=h-th-12: \
       shadowx=1:shadowy=1:shadowcolor=black@0.4: \
       line_spacing=3" \
  -c:v libx264 -preset fast -crf 18 \
  -c:a copy \
  output.mp4
```

### 命令参数速查

| 参数 | 作用 | 计算公式 |
|------|------|---------|
| `fontfile` | 字体路径 | 固定 `resource/fonts/STHeitiMedium.ttc` |
| `text` | 双行文本 | `归属行\n地址行` |
| `fontsize` | 归属行字号 | `max(12, round(h × 0.022))` |
| `fontcolor` | 文字颜色+透明度 | `white@0.85` |
| `box` | 背景框开关 | `1` |
| `boxcolor` | 背景颜色+透明度 | `black@0.35` |
| `boxborderw` | 背景框内边距 | `round(font_size × 0.4)` |
| `x` | 水平位置（右对齐） | `w-tw-right_margin` |
| `y` | 垂直位置（底对齐） | `h-th-bottom_margin` |
| `shadowx/y` | 阴影偏移 | `1:1` |
| `shadowcolor` | 阴影颜色+透明度 | `black@0.4` |
| `line_spacing` | 行间距 | `max(2, round(font_size × 0.15))` |
| `-c:a copy` | 音频流直通 | —（零损失） |

## 八、质量保障

### 可读性检查清单

- [ ] 白色文字在浅色视频背景上可见（背景框保证对比度）
- [ ] 文字不超出视频边界（`x=w-tw-margin` 自动右对齐）
- [ ] 水印不遮挡关键内容区域（右下角常规为次要区域）
- [ ] 不同分辨率下水印比例协调（自适应公式保证）
- [ ] 中/英文切换后视觉效果一致（统一字体和配色）

### 边缘情况

| 场景 | 处理策略 |
|------|---------|
| 视频极短（< 3s） | 水印覆盖全时长，`enable='between(t,0,9999)'` |
| prompt 含中英混合 | 优先判断含 CJK → 使用中文水印 |
| prompt 为空 | 默认“英文”水印 |
| 自定义分辨率 | 公式自适应，最小字号保护 12px |
| 字体文件缺失 | 记录 warning 日志，跳过水印（不阻塞管线） |
